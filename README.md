## Configuration Steps
1. Cloning the repository:

```
$ git clone https://github.com/sonic-sabers/Headout_Clone.git
```

2. Open the project:

`$ cd Headout_Clone`

3. Now run the app on your connected device (using terminal):

`$ npm install` then
`$ npx react-native run-android`
